# mChat
> Lightweight, simple and up to date chat plugin for Minecraft.

This plugin adds local and global chat, also roleplay commands like /me.

![](logo.png)

## Requirements

- Java 17+
- PlaceholderAPI (optional)
- Vault (optional)
